How are you using Glassboard?

We use Glassboard for every conference we organize, often creating two boards, a speakers only board that allows speakers and myself to coordinate things, plan, and in general chat. The other board is the overall conference board that every attendee is encouraged to join. Glassboard has become the official unofficial back channel for the conference. Whether it’s attendees planning a night out or which session to hit next, it’s a great way for everyone to be able to communicate.

Why Glassboard?

We started using Glassboard because it was created by friends of the conference, but likely would have started using it once we heard about it anyway. Twitter is great, but is not private, we like that glass board boards are private so attendees can talk about sessions and speakers talk about things that aren’t relevant to attendees.

If not Glassboard what would you use?

Honestly I don’t know. probably something more broadcast. Our official conference app allows us to push updates to attendees, that plus twitter would likely have to fill the gap.
